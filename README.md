# xeve
[xeve]是一个客户端/服务器网络库

## 依赖:

1. [evpp]
2. [glog]
3. [gflags]
4. [gtest] (optional)
5. [protobuf]
6. [mongo-cxx-driver] (optional)
7. [libgo] (optional)
8. [cpp_redis] (optional)

## 编译工具:
### linux平台:
1. gcc (GCC) 4.8+
2. GNU Make
3. CMake
###  windows平台:
1. [Visual Studio 2017]
2. 使用[vcpkg]安装`evpp`,`glog`,`gflags`,`gtest`,`protobuf`,`mongo-cxx-driver`,`libgo`,`cpp_redis`
